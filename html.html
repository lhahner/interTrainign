HTML - Cheat Sheet Day 1

Liste der behandelten HTML befehle

  • Anchor: <a href="url">Link<a>
  • Image: <a href="url">Link<a>
  • Paragraph: <p>Link<p>
  • Main: <main></main>
  • Nav: <nav></nav>
  • Footer: <footer></footer>
  • Headings:
    1. Heading 1: <h1></h1>
    2. Heading 2: <h2></h2>
    3. Heading 3: <h3></h3>
    4. Heading 4: <h4></h4>
    5. Heading 5: <h5></h5>
    6. Heading 6: <h6></h6>
  • Liste:
    1. unorderd list: <ul><li></li></ul>
    2. ordered list: <ol><li></li></ol>
  • Table: <table> <tr> <th></th> </tr> <tr> <td></td> </tr> </table>
